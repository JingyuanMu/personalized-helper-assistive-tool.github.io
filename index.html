Abstract

Personalized Helper and Assistive Tool aims to address the privacy and efficiency challenges of virtual assistants by developing a local Large Language Model (LLM) solution. Unlike traditional virtual assistants like Alexa and Google Assistant, which often compromise user privacy and demand significant computational resources, our approach leverages a local LLM to process user prompts and manage conversations securely on-device. The system interacts with a local database containing contacts and documents, enabling the AI to perform tasks such as sending emails, creating calendar events, summarizing PDFs, and searching the internet via an API. By minimizing reliance on external servers and optimizing for low computational overhead, this project ensures enhanced user privacy and efficient task execution, making virtual assistants more accessible and secure for everyday use.

System Overview: The structure of the PHAT virtual assistant system is designed to operate locally, ensuring privacy and efficiency. At its core, the PHAT model processes user prompts to execute various tasks. It integrates with a local database containing contacts and files (e.g., PDFs) to manage user data securely. The system supports multiple functionalities: sending emails, extracting text from documents, searching the web via an API, and scheduling events. Each function is modular, allowing the PHAT model to interact with specific components as needed, such as accessing contacts for email tasks or files for text extraction, all while maintaining on-device processing to minimize external dependencies.
