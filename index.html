Protein Master

An Intelligent Platform for Protein Structure Retrieval and Visualization

Jingyuan Mu, Xiaoyu Hu, Haolun Zhan, Hanzhong Liu
Texas A&M University
Code Paper

Abstract

Protein Master is a web-based system for visualizing and analyzing protein structures through intuitive 2D and 3D representations. It enables researchers to explore spatial relationships between amino acids and understand structural patterns across dimensions. Key features include synchronized interaction between 2D contact maps and 3D conformations, atom-level exploration, and dual-protein comparison. The platform integrates a scriptable query editor and an AI assistant to simplify complex structural searches, allowing users to generate and execute queries using either code or natural language.

System Overview: Protein Master offers a modern, extensible interface for visualizing protein structures in both 2D and 3D. It provides real-time interaction across views and supports natural language queries via LLM integration.
